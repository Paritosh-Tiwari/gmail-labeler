Install the bookmarklet

Make sure your bookmark bar is visible (Ctrl+Shift+B in Chrome / Edge), then drag this link onto it:

Label this Email

Pasting the URL directly into Chrome's address bar won't work — Chrome strips the javascript: prefix as a security measure. Drag-to-bookmark-bar is the only reliable install.

How to use

  1. In Gmail, open any specific email (URL changes to ...#inbox/FMfcgz...).
  2. Click the Label this Email bookmark.
  3. A new tab opens here with a proposal.
  4. Tweak the suggested label / filter, click Apply.
