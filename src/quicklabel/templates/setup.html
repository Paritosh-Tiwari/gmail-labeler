Create a Google Cloud OAuth client

You'll do these in your Google account. Sign in as the Gmail address you want QuickLabel to manage.

  1. Create a new Google Cloud project. Open project creator ↗
    Name it anything (e.g. QuickLabel). No billing setup needed.
  2. Enable the Gmail API. Gmail API page ↗
    Make sure your new project is selected in the top dropdown, then click ENABLE.
  3. Configure the OAuth consent screen. Consent screen ↗
    User type: External. Fill in app name + your email. Add yourself under Test users. Required scopes: https://www.googleapis.com/auth/gmail.modify and https://www.googleapis.com/auth/gmail.settings.basic
  4. Create OAuth client ID. Credentials page ↗
    CREATE CREDENTIALS → OAuth client ID. Application type: Desktop app (very important — Web application won't work). Name it anything. Click CREATE → DOWNLOAD JSON.
